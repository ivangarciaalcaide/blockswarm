create_genesis_block(self)

source code 

Creates first block of the chain and append it to the chain.

Considerations:

  • It is indexed at position 0.
  • As there is not a previous block, previous_hash is set to "0".
  • As there are not transactions to mine yet, transaction list is empty.

proof_of_work(self, block)

source code 

Proof of work implementation.

It takes a block and sets its nonce to 0. Then, calculates its hash from Block.get_string_to_hash(self). This hash must end with, at least, as many '0's characters as indicated in pow_difficulty to success. I hash doesn't accomplish PoW, nonce value is incremented by 1 and it starts over.

Once nonce is found, block's hash is updated.

Parameters:
  • block - Block over the proof of work is executed. # @return: String with the hash that accomplish the PoW.

mine(self, allow_empty_transactions=False)

source code 

Mines a new block and add it to the chain.

Mining steps:

  1. Selects transactions to be incorporated to the block.
  2. Creates a block with current information available
    • Selected transactions
    • Current time for timestamp
    • Previous hash filled with last block hash.
  3. Fills hash field with proof of work.
  4. Removes mined transactions from unconfirmed_transactions list.
Parameters:
  • allow_empty_transactions (boolean) - True, mine the block even if transaction's block list is empty.
Returns:
False, If block could not be mined. The mined block, otherwise.

is_valid_block(self, block)

source code 

Checks if the block is a valid one.

To be a valid block it has to accomplish following criteria:

  1. Its previous hash must be the same as previous block hash.
  2. Its index must be the following from previous block.
  3. Its hash must end with at least pow_difficulty numbers of '0's
  4. Its hash must be the same than calculated hash.
Parameters:
  • block - Block to be checked
Returns:
False if not valid. True, otherwise.
